Posibilities of brackets expressions : YOU CAN DO

expressions:

2+2= {{2+2}}

concatening string with properties:

Welcome {{name}} method 1

{{' Welcome ' + name}} method 2

access to a JavaScript methods:

name.length : {{name.length}}

name.toUpperCase() : {{name.toUpperCase()}}

helloWorld() : {{helloWorld()}}

YOU CAN'T DO

in brackets : a=2+2 -->> uncomment and see the console

in brackets : window.location.href -->> uncomment and see the console

We can do: create a variable on the class instance and interpolate here : example

{{siteUrl}}

where the variable siteUrl = window.location.href
